# Crochet Store E-Commerce Website 🧶

## About The Project
This is an e-commerce crochet store website developed as a major BSc IT project.

The website allows users to:
- Browse crochet products
- Login/Register
- Search products
- Place orders
- Submit reviews

---

## Technologies Used
- HTML
- CSS
- PHP
- MySQL

## Features
- User Authentication
- Product Search
- Order System
- Review System
- Responsive Design
